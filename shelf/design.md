# ディスプレイ棚 設計書

## 要件

- 机: 幅100cm × 奥行55cm × 高さ70cm
- ディスプレイ: 24インチ × 2台（各約5kg、合計約10kg）
- 手前にノートPCがあり、ディスプレイと被るのを解消したい
- 棚は机の背面に密着させ、床から置く独立した家具
- 棚の天板高さ: 床から75cm（机より5cm高い）
- 棚の奥行き: 23cm
- 棚の幅: 108cm（内寸104.4cm）。机(100cm)を棚の中に差し込んで奥行きを調整できるようにする

## 設計

### 配置イメージ（横から見た図）

```

                        │ディスプレイ│
       ┌─────┐         └────┬─────┘
       │ノートPC│    ┌──┬─────┴─────┐ ← 床から75cm
       └──┬──┘  │補強│  棚天板   │
  ────────┴───────┐材│          ┃│ ← 背板(4mm)
    机の天板 70cm  │  │          ┃│
                  │  │   脚     ┃│
                  │  │          ┃│
  ────────────────┘  └──────────┘
                  ├23cm┤
        ← 机 55cm →

※天板裏に補強材（角材30×18mm）を1本貼り付け
※背面に薄ベニヤ(4mm)を全面貼り付け → 横揺れ完全防止
※机の天板(70cm)と棚天板裏面(73.2cm)の間に3.2cmの余裕があり干渉しない
```

### 寸法

| 部品       | 幅       | 奥行/高さ | 厚さ   | 数量 |
|------------|----------|-----------|--------|------|
| 天板       | 108cm    | 23cm      | 1.8cm  | 1枚  |
| 側板       | 23cm     | 73.2cm    | 1.8cm  | 2枚  |
| 背面横板(上)| 104.4cm | 7.5cm     | 1.8cm  | 1枚  |
| 背面横板(中)| 104.4cm | 7.5cm     | 1.8cm  | 1枚  |
| 背面横板(下)| 104.4cm | 7.5cm     | 1.8cm  | 1枚  |
| 補強材(角材)| 104.4cm | 3.0cm     | 1.8cm  | 1本  |

### 寸法の根拠

- **天板幅 108cm**: 側板(1.8cm×2)を引いた内寸が104.4cm。机(100cm)が両側約2cmの余裕をもって入る
- **奥行 23cm**: ディスプレイスタンドが載る奥行き
- **側板の高さ 73.2cm**: 天板厚(1.8cm)を足して合計75cmになる
- **背面横板 3本**: 上・中・下に配置して横揺れを防止。下5cmは開ける
  - 上: 天板直下（天板との接合部を補強）
  - 中: 高さ中央付近（床から約35cm）
  - 下: 床から5cmの位置（底面から5cm開ける）
- **補強材**: 天板裏面の中央（奥行方向の中心）に横方向に1本貼り付け。天板裏面の高さは床から73.2cm、机の天板は70cmなので、補強材(高さ3cm)を付けても床から70.2cmとなり、机の天板とほぼ同じ高さで干渉しない

### 強度について

- 天板スパン約104cm(側板間の内寸)に対し、10kgの荷重
- 天板裏に補強材(30×18mm角材)を1本貼ることで、断面がT字型になり剛性が大幅に向上
- たわみは実質ゼロに近くなる
- 背面横板3本が横揺れを防止。3本あることで2本よりも大幅に安定
- 壁に密着させることでさらに安定

※250mm幅の集成材を23cmにカットして使用。

**カット計画:**
- 集成材: 2枚とも250mm → 230mmに縦カット
- 1枚目(1820mm) → 天板(1080mm) + 側板(732mm) = 1812mm ✓ 端材8mm
- 2枚目(1820mm) → 側板(732mm) + 背面横板元板(1044mm) = 1776mm ✓ 端材44mm
  ※背面横板は230mm幅の板(1044mm)を縦に3分割(約7.5cm幅 × 3本)
- 角材(1820mm) → 補強材(1044mm) ✓ 端材776mm

### 組み立て方法

1. 集成材を寸法通りにカット（ホームセンターのカットサービス推奨）
2. 250mm幅 → 230mmに縦カット
3. 背面横板用に230mm幅の板(1044mm)を縦に3分割カット(約7.5cm幅 × 3本)
4. 角材を1044mmにカット
5. サンドペーパーで切り口を整える
6. **補強材を天板の裏面中央に木工用ボンドで貼り付け、乾燥させる**
7. 側板の上端にボンドを塗り、天板を載せてネジ止め（各2本）
8. L字金具で天板と側板の内側を補強
9. 背面横板(上)を天板直下の位置に、側板の背面にネジ止め（天板にもネジ止めすると補強になる）
10. 背面横板(中)を床から約35cmの位置に、側板の背面にネジ止め
11. 背面横板(下)を床から5cmの位置に、側板の背面にネジ止め
12. はみ出たボンドを拭き取り、乾燥させる

### オプション

- **滑り止め**: 脚の底面にゴム足やフェルトを貼ると、床を傷つけない
- **転倒防止**: L字金具で壁に固定すると地震対策になる
- **塗装**: ワックスやニスで仕上げると見栄えが良くなる
- **配線**: 背面横板の間の隙間からケーブルを通せる
