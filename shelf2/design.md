# 玄関 目隠し収納棚 設計書

## 要件

- 玄関正面、既存の飾り棚（奥行40cm）の下、床〜飾り棚下面 **106cm** の空間に置く
- 開口幅120cmのうち、左端に自動掃除機ステーション（幅32cm）が居座るため、棚はその右側に置く
- **内幅は44cm**（開口の残り88cmを全部使い切らず、掃除機の出入りに余裕を残す）
- 奥行きは **40cm**、上の飾り棚と前面を面一に揃える
- **全高は90cm**
- 収めるもの
  - シュレッダー（幅20cm、**高さ33cm**、棚に入れたまま使う／扉を開けて上から紙を投入する）
  - 充電アダプター、電源タップ → **シュレッダーと同じ段の横に並べる**
  - 道具箱 40 × 32 × 10cm
- **掃除機のアタッチメント・消耗品は、棚の天板の上に置く**（棚の中には入れない）
- **扉は1枚（片開き）** で正面全体を覆う。閉めれば玄関正面から中身が見えない
- 電源タップのコードを外に出せること（背面から）

### 寸法の根拠

- **奥行400mm**: 上の飾り棚（奥行40cm）と前面を揃えるため。同時に道具箱の奥行32cmを寝かせて収める条件でもある（30cmだと入らない）。**幅400mmの集成材をそのまま使えるので、本体側は縦カットが一切発生しない**。
- **内幅440mm**: 開口120cmから掃除機ステーション32cmを引いた88cmを全部使わず半分に留め、掃除機の出入りに余裕を残す。道具箱の幅40cmが余裕4cmで収まる寸法でもある。
- **外幅476mm**: 440 + 側板18×2。掃除機320 + 棚476 = 796mm で、開口1200mmに対し404mmの余裕。
- **全高900mm**: 使える高さ1060mmに対し160mmの余裕。**この寸法は木取りと噛み合っている** — 側板900 + 棚板440 + 棚板440 = 1780mm で、規格材1820mmにちょうど収まる。
- **シュレッダー段706mm ← 設計上いちばん重要な寸法**: シュレッダーの高さ330mmに対し、**上に376mmの空間**を確保している。A4用紙を手で持って上から投入口に差し込むには、紙の長さ（297mm）に手指のぶんを足した空間が要る。**当初案の400mm（頭上70mm）では紙を差し込めず、「棚に入れたまま使う」という要件が成立しなかった。** 中段（籠段）を廃止し、その306mmをまるごとこの空間に振り替えている。
  - シュレッダーの投入口は床から約506mmの高さにくる。立ったまま少しかがむ姿勢で投入できる。
- **シュレッダー段の横幅配分**: 内幅440 − シュレッダー200 = **240mm** が電源タップと充電アダプターの置き場。一般的な6口タップ（幅約40〜50mm × 長さ300mm）は奥行き方向に寝かせて置けば余裕で収まる。
- **底段140mm**: 道具箱の高さ100mm + 出し入れの指が入る余裕40mm。
- **消耗品を天板の上に置く**: 棚の中に籠段を設けると、その天井がそのままシュレッダーの投入を邪魔する。棚の上（床から900mm）なら、上の飾り棚まで160mmあるので、掃除機の紙パックやアタッチメントを平置きできる。**棚内部の高さを一切食わないのが利点。**
- **扉 476×900×15、238mm幅の2枚接ぎ**: 幅は側板の外側までかぶせる「外かぶせ」なので外寸476mmと同じ。高さは床から天板の下端まで、正面を丸ごと覆う。
  - **なぜ一枚板でなく2枚接ぎか**: 900mmは規格材1820mmのちょうど半分なので、**1820×250×15 を1枚買えば 900×250 が2枚取れる**（3,828円、端材20mm）。一枚板でやるには 1820×500×15 が必要で、面積が倍なので**7,000円台と見込まれ、しかもその半分（端材920×500）を丸ごと捨てることになる**。集成材の実売価格が想定より高かったぶん、この判断の効果は大きい（3,000円以上の差）。250mm幅は集成材でもっとも一般的なサイズで、在庫を切らしている店がまずないという入手性の利点もある。
  - **反りにも強くなる**: 木は幅方向に伸縮するので、幅238mmの板は幅476mmの板より絶対変位が半分。そこへ裏桟が横方向のつっぱりとして入るため、一枚板より確実に平らを保つ。15mmという厚さを選んだ理由（反り止め）が、より確実な方法で達成されている。
  - **238 × 2 の対称割りにした理由**: 継ぎ目は正面から見える。片方だけリップして 250 + 226 にすればカットが1回減るが、継ぎ目が偏って目立つ。2回リップして中央に持ってくるほうが自然に見える。
- **扉の裏桟 430 × 40 × 18 を3本 ← 長さが重要**: 外かぶせの扉は**裏面が側板の前小口に当たって閉まる**ので、裏桟が476mm幅いっぱいにあると桟が側板にぶつかって扉が18mm浮く。桟は**内寸440mmの中に収まる長さ（430mm、左右5mmずつの逃げ）**にして、閉めたとき桟が棚の中に入り込むようにする。
- **裏桟の高さ 60〜100 / 420〜460 / 800〜840（床から）**: 桟は閉めたとき棚の中へ18mm突き出す。**底板(0〜18)・棚板1(158〜176)・天板(882〜900) と同じ高さにあると干渉して閉まらない**ので、その3箇所を避けた位置に配置する。上・中・下の3本にすることで、900mmある継ぎ目を均等に押さえられる。
  - この18mmの出っ張りのぶん、**シュレッダーと道具箱は前面から18mm以上奥に置く**こと。奥行き400mmに対しシュレッダーは250〜300mm程度なので、実用上の余裕は十分ある。
- **側板を18mmのままにした理由**: 荷重ではなく蝶番のため。外かぶせの扉は蝶番の片羽を側板の前面小口に留めるので、小口にネジ2本を並べられる厚みが要る。12mmだと1本芯打ちが精一杯で、そこがガタつきの起点になる。

### 強度について

- **棚板スパンは440mm** と短い。18mm厚のパイン集成材はこのスパンなら10kg程度を載せてもたわみは目視できないレベル。シュレッダー（約4〜6kg）、道具箱（工具込みで5kg前後）はいずれも余裕。**棚受け金具や補強桟による棚板の補強は不要**（過剰設計になる）。
- **天板の上に物を載せる前提だが問題ない。** スパン440mmに対し掃除機の紙パックや小物では数kgがせいぜい。ただし**天板の上は「置き場」であって「載せ台」ではない**ので、重い工具箱などをここに積むのは避ける。
- **⚠ 横揺れ（ラッキング）対策が最重要になった。** 中段を廃止したことで、側板同士をつなぐ水平材が **底板・棚板1・天板の3枚だけ** に減った。四角形が平行四辺形に潰れるのを止める要素が減っているので、**背面の桟3本（上・中・下）は絶対に省略しないこと。** 当初案では「入れたほうがよい」補強だったが、この構成では**必須**に格上げされる。
  - 桟を薄くしても意味はない。桟にかかる力は材の長さ方向で、効いているのは断面の40mm側。30mmという厚みを削っても剛性はほぼ変わらず、逆にネジ止めで割れやすくなるだけ。
  - 桟の「中」の1本は、シュレッダー段のちょうど中ほど（床から約500mm）に入れる。ここが一番長い無支持区間の真ん中で、効きがいちばん大きい。
- **側板の無支持区間は706mm** に伸びた。ただし側板は奥行き400mmという深い断面を持つため、前後方向にはきわめて曲がりにくい。左右方向へのふくらみは背面の桟と天板が押さえる。実用上の問題はない。
- **転倒対策**: 幅476に対し高さ900（比率1:1.9）で、それ自体は倒れにくい。ただし扉を開けると重心が前に出るので、壁側にぴったり寄せて設置する。
- **扉の重量は約3.7kg**（板3.2kg + 裏桟3本0.5kg）。**平蝶番は3個**。2個でも保持はできるが、片開き扉は上下2点だと数年で先端が垂れて閉まらなくなる。上・中・下の3点で支える。
- **蝶番のネジは扉板の15mmに効かせる。** 裏桟は430mmで扉の左右端（各23mm）には届かないので、**桟と蝶番の位置を合わせても補強にはならない**。ネジは12mm程度の短いものを選び、扉の表面に突き抜けないようにする。
- **扉の反りと継ぎ目**: 2枚接ぎの継ぎ目は、木端をボンドで接着したうえで裏桟3本が横方向を拘束するので、開くことはない。むしろ**幅の狭い板2枚＋桟のほうが、幅476の一枚板より反りにくい**。加えてマグネットキャッチを**上下2箇所**（天板の裏と棚板1の裏）に付けると、閉まりも安定する。

**参考：もし総額を下げたい場合**

メルクシパイン集成材は 1820×400×18 で 6,578円 = **約9,000円/m²**。これに対し**シナ合板やラワン合板は同じ18mm厚で 5,000〜6,000円/m² 程度**と、面積単価が3〜4割安い。

しかも 1820×910 の合板1枚（約9,000〜10,000円）から、本体の全部品が取れる：

```
1820×910 の板を幅方向に 400 / 400 / 110 に割る（縦カット2回）
  400幅① : 側板900 + 棚板440 + 棚板440 = 1780 ✓
  400幅② : 側板900 + 棚板440 = 1340 ✓
  110幅   : 端材
```

**集成材2枚（13,156円）が合板1枚（約9,000円）に置き換わり、4,000円ほど下がる。** ただし引き換えに以下を受け入れることになる。

- **小口に積層の縞が出る** — 玄関正面に置く家具なので、木口テープ（約500円）を貼る手間が増える
- **縦カットが2回必要** — 910mm幅の板を割るので、店のカットサービス次第では追加料金
- **1820×910の板は運搬が大変** — 軽自動車には積めない

見た目を優先するなら集成材のまま、コストを優先するなら合板。**本設計は集成材を前提に書いている。**

**カット計画:**

本体は奥行き方向に板幅400mmをそのまま使うので**縦カットゼロ**。扉の2枚を238mmに詰める縦カットが2回入る。

- **集成材A（1820×400×18）**: 側板 900 + 棚板 440 + 棚板 440 = **1780mm** ✓ 端材 40mm
- **集成材B（1820×400×18）**: 側板 900 + 棚板 440 = **1340mm** ✓ 端材 **480×400**
- **集成材C（1820×250×15、扉用）**: 900 + 900 = **1800mm** ✓ 端材 20mm
  - さらに **各板を 250 → 238 にリップ**（12mmずつ落とす、縦カット2回）
- **角材（30×40×1820）**: 440 × 3本 = **1320mm** ✓ 端材 500mm
- **扉の裏桟**: 集成材Bの端材 480×400 から **430 × 40 を3本**（480→430に横カット、残りを40mm幅にリップ）

440×400 の板は A から2枚、B から1枚で **計3枚**。底板・棚板1・天板でちょうど使い切る。

Aはのこ代（3カット × 約3mm = 9mm）を足しても1789mmで1820mmに収まるが、**端材40mmでほぼ使い切りなので、カット指示は「900を1本、440を2本」と明確に伝えること**。長さの取り違えが起きると板が1枚足りなくなる。

Cも 900 + 900 = 1800mm と端材20mmでぎりぎり。**横カットは1回だけ**なのでのこ代3mmを引いても収まるが、板の端が直角に切れているか（工場断面が斜めでないか）を店で確認しておくと安全。

Bの端材480×400は、裏桟3本（430×40 が3本 = 幅120mm分）を取ってもまだ大きく残る。シュレッダー段の右側に小棚を足したくなったときの材料になる。

### 組み立て方法

1. **カットを受け取ったら寸法を検品する。** 特に440mmの板3枚が揃っているか。1枚でもずれていると箱が捻れる。
2. **全部品の角と小口を紙やすりで整える**（#180 → #240）。玄関正面に置くので、扉の角と天板の前縁は特に丸める。組んだ後だと内側にやすりをかけられないので、必ずこの段階で。
3. **側板に棚板位置の墨線を引く。** 各棚板の**下面**の高さで引く。床側を基準に **0（底板・側板の下端に合わせる）/ 158（棚板1）/ 882（天板・側板の上端から18mm下）**。**2枚の側板を並べて同時に引く**こと。片方ずつ引くとずれる。
4. **下穴をあける。** 側板の外側から、各棚板の厚み中心（墨線+9mm）に向けて、1枚あたり3箇所（前・中・後）。集成材は下穴なしでネジを打つと割れる。
5. **底板から順に組む。** 棚板の小口にボンドを塗り、側板を当てて45mm木ネジで固定。片側の側板に3枚（底板・棚板1・天板）を全部付けてから、もう一方の側板をかぶせると作業しやすい。
6. **直角を確認する。** ボンドが乾く前に、対角線の長さを2方向測って同じになるよう調整する。**水平材が3枚しかないぶん、この段階では箱がまだ柔らかい。** ここが最後の修正チャンス。
7. **背面の桟を3本取り付ける。** 側板の後ろ端に、上（天板のすぐ下）・中（床から約500mmの高さ）・下（底板のすぐ上）へ角材を渡し、30mm木ネジで固定。**この工程で箱の剛性が決まる。** 各端2本ずつ、計4本のネジでしっかり留める。
8. **シュレッダーを実際に入れて確認する。** 扉を付ける前にやること。**紙を持って上から投入口に差し込む動作が無理なくできるか**を、実際の紙でやってみる。ここで問題が出たら、扉を作る前なので棚板1の位置を下げてやり直せる。
9. **扉のパネルを作る。** 238×900の板2枚を平らな床（または作業台）の上に、**裏面を上にして**並べる。
   - 合わせる木端にボンドを塗り、2枚をぴったり突き合わせる。**表面側に段差が出ないよう、平らな面に伏せて作業する**のがコツ。
   - 裏桟3本（430×40×18）を、**床から 60〜100 / 420〜460 / 800〜840** の高さに横向きに渡す。左右は扉の中央に揃える（扉幅476に対し桟430なので、両端に23mmずつ余る）。
   - 桟にボンドを塗り、30mm木ネジで留める。**ネジは1本の桟につき4本（各板に2本ずつ）**、継ぎ目をまたぐように配置する。扉板は15mmなので、**ネジが表に突き抜けないよう必ず下穴をあけ、長さを確認する**。
   - ボンドが乾くまで（1時間程度）そのまま置く。重しを載せると継ぎ目が締まる。
10. **扉を棚に当ててみる。** 蝶番を付ける前に、扉を閉じる位置に手で当て、**裏桟が棚の中にすっと収まるか**を確認する。桟が側板や棚板に当たって扉が浮くようなら、桟の位置か長さを直す。ここを飛ばすと蝶番を付けた後に判明して手戻りになる。
11. **扉に蝶番を付ける。** 扉の裏面（吊り元側の端）に平蝶番3個を、上端から100mm・中央・下端から100mmの位置に取り付け、側板の前端小口に留める。小口へのネジなので**必ず下穴をあける**こと。扉側は15mm厚なので、**12mm程度の短いネジ**を使って表に突き抜けないようにする。**開き勝手は、掃除機ステーションと反対の右側を吊り元にする**（左に開くと掃除機側にぶつかる）。
12. **マグネットキャッチ2個と取っ手を付ける。** キャッチは吊り元と反対側の、天板の裏と棚板1の裏へ。
13. **設置して動作確認。** 扉を閉めた状態で隙間が均一か、シュレッダーが実際に使えるか、コードが背面から無理なく出ているかを確認する。**シュレッダーと道具箱は、裏桟の出っ張り（18mm）を避けて前面から18mm以上奥に置く。**

### オプション

- **電源まわりの小棚**: シュレッダー段の右側240mm幅は上方向が大きく空く。端材480×400から仕切り板と小棚を作れば、充電アダプターを2段に整理できる。ただし**シュレッダー側（左200mm幅）の上方は絶対に塞がないこと** — 紙の投入空間そのものなので。
- **転倒防止**: 背面上部の桟を通して壁の下地にL字金具で固定する。桟があるのでネジが効く。
- **底のフェルト/滑り止め**: 床の傷防止と、掃除機ステーションを動かすときに一緒にずれるのを防ぐ。
- **塗装**: 玄関の既存家具に色を合わせるなら、オイルステイン or 水性ウレタンニス。**組み立て前に塗ったほうがきれいに仕上がる**が、接着面には塗らないこと。扉は15mmと薄いので、**片面だけ塗ると反るため必ず両面塗る**。
- **配線**: 電源タップはシュレッダー段の右奥に置き、コードは背面の開口から出す。タップを側板や背面桟にマジックテープで浮かせて留めると、埃がさらに溜まりにくい。
- **天板の上**: 消耗品を置くスペース（飾り棚まで160mm）。玄関正面から見える位置なので、蓋つきのケースや籠にまとめると見た目が整う。

## 未確定事項（作る前に実測してください）

- **シュレッダーの奥行き**。高さ33cmは確認済みだが、奥行きは未確認。設計は400mm以内を前提にしている。ダストボックスを手前に引き出すタイプなら、扉を開けた状態での引き出しスペースも要確認。
- **シュレッダーの投入口の位置**（天面の前寄りか奥寄りか）。奥寄りだと、手を伸ばす角度がきつくなる。工程8で実際の紙を使って確認する。
- **床から飾り棚下面までの1060mm**。左右で床が傾いていることがある。90cmなら160mmの余裕があるので問題になりにくいが、天板の上に物を置く前提なので実測しておきたい。
- **15mm厚 1820×250 の在庫**。250mm幅は最も一般的なサイズなので在庫リスクは低いが、15mm厚は18mmより品揃えが薄い店もある。なければ18mm厚で作る（扉が3.9kgになるが、蝶番3個なら許容範囲）。
