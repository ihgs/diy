# 基本設計

## 背景

玄関正面に奥行き40cmの飾り棚があり、その下に自動掃除機ステーションや、シュレッダー、充電アダプター、電源タップなどがある。
玄関の正面であり、見た目がよくない、電源タップに埃が溜まりやすいという問題がある。
シュレッダー、充電アダプター、電源タップなどを覆い隠す棚を作りたい。

## イメージ

rough.drawio.png を参照

## 追加要望

- 40cm x 32cm x 10cm の道具箱も入れたい
- 電源タップへのコンセントを刺すための穴をどこかにあけたい(後ろからでもOK)
- 扉をつけたい
- 掃除機のアッタッチメントや、消耗品を入れるところもほしい（上部に籠でもOK）
- シュレッダーは、棚に入れたままつかいたい
