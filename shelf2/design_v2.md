# 玄関 目隠し収納棚 設計書 v2（パイン棚柱を使う案）

**v1（`design.md`）は残してあります。** こちらは「メルクシパイン集成材18ミリ専用支柱（パイン棚柱）」を使って、側板を構造材から外した案です。末尾に v1 との比較表があります。

## ⚠ 先に確認が必要なこと

本設計は下記を前提にしています。**支柱の実物を見て確認してから買ってください。** 特に溝の位置は段の高さを直接決めるので、ここがずれると設計全体が動きます。

| 確認項目 | 本設計の前提 | 確認できた根拠 |
|---|---|---|
| 断面寸法 | 27 × 60mm | 他社同等品の実測値 |
| 溝の高さ | 18mm（集成材18mm厚が入る） | 商品名「18ミリ対応」＋他社同等品 |
| 溝の奥行き | 15mm | 他社同等品の実測値 |
| 溝のピッチ | **約80mm** | 他社同等品（1470mm・溝15ヶ所）からの換算 |
| **900mm長の在庫と価格** | **1本 約1,100円** | **DCM同等品の実売（ビバホーム品は未確認）** |
| 片溝／両溝 | **片溝**（溝が内側だけ） | 両タイプが存在。**要指定** |
| 最上・最下の溝の位置 | 端から40mm程度 | **未確認。段の高さに直結する** |

**溝の位置が想定とずれても設計は破綻しません。** 段の高さを一番近い溝に合わせるだけです（それが可動棚の利点）。ただし材料費と扉の高さは変わる可能性があります。

## 要件

v1 と同じ。変更なし。

- 玄関正面、既存の飾り棚（奥行40cm）の下、床〜飾り棚下面 **106cm** の空間に置く
- 開口幅120cmのうち、左端に自動掃除機ステーション（幅32cm）。棚はその右側
- 奥行きは **40cm**、上の飾り棚と前面を面一に揃える
- **全高は90cm**
- 収めるもの
  - シュレッダー（幅20cm、高さ33cm、**棚に入れたまま使う**／扉を開けて上から紙を投入）
  - 充電アダプター、電源タップ → シュレッダーと同じ段の横
  - 道具箱 40 × 32 × 10cm
- 消耗品・アタッチメントは天板の上に置く
- 扉は1枚（片開き）
- 電源タップのコードを背面から出す

## v2 の考え方

v1 では **側板（1820×400×18 の集成材2枚）が構造そのもの**でした。棚板をネジで側板に留め、側板が荷重を受ける設計です。集成材が13,156円と材料費の6割を占めていました。

v2 では **4本の支柱が構造を担い、棚板は溝に差し込むだけ**にします。側板は荷重を受けないので、あってもなくてもよくなります。

これによって変わるのは主に3点です。

1. **集成材が2枚 → 1枚**（棚板3枚は1枚から取れる）
2. **棚板が可動になる**（80mmピッチで位置を変えられる）
3. **棚板のネジ止め・墨線・下穴が不要になる**（溝に差し込んでビス穴から留めるだけ）

### 寸法の根拠

- **棚板の幅470mm ← v2でいちばん間違えやすい寸法**: 支柱の溝は**片側15mm**あるので、棚板は左右の溝へ15mmずつ差し込まれる。したがって **棚板幅 = 内寸440 + 15 × 2 = 470mm**。v1の440mmのまま発注すると、左右に15mmずつ隙間が空いて棚板が落ちる。
- **内寸440mm**: v1と同じ。道具箱の幅400mmが余裕40mmで収まる。
- **外寸幅494mm**: 440 + 支柱27 × 2。v1（476mm）より18mm太るが、掃除機320 + 494 = 814mm で開口1200mmに対し386mmの余裕があり問題にならない。
- **奥行400mm**: 支柱（60mm幅）を前後に2本ずつ置き、その間隔を含めて棚板の奥行を400mmにする。支柱は前0〜60、後340〜400の位置。上の飾り棚と面一のまま。
- **支柱は4本（前後×左右）**: 棚板は左右の溝で支えられるが、1本ずつだと棚板の前後どちらかしか押さえられない。前後2本ずつで棚板の前縁・後縁の両方を保持する。棚板の中央280mmは溝がないが、スパンとしては短く問題にならない。
- **段の高さは「溝の位置」で決まる。** 下記は**最下段の溝が床から80mm・ピッチ80mm**と仮定した割り付け。溝は 80 / 160 / 240 / … / 880 に並ぶ。**実物の溝位置を測って、一番近いところを選び直すこと。**

  | 部材 | 溝の位置（下面） | 段の高さ |
  |---|---|---|
  | 底板 | **80**（最下段） | 支柱の足元 80mm |
  | 棚板1 | **240**（80×3） | 底段 = 240 − 98 = **142mm** |
  | 天板 | **880**（80×11） | シュレッダー段 = 880 − 258 = **622mm** |

- **底段142mm**: 道具箱の高さ100mm + 出し入れの余裕42mm。**溝1つ下（160mm）にすると底段が62mmになり道具箱が入らない**ので、ここは240の溝を使う必要がある。
- **⚠ シュレッダー段622mm・頭上292mm ← v1からの唯一の実質的な後退**: シュレッダー330mmの上に292mmが残る。v1は706mm段・頭上376mmだったので、**84mm狭くなっている**。A4の長辺が297mmなので、紙を垂直に立てて持つと少し当たる。斜めに差し込めば問題なく入るが、v1ほどの余裕はない。
  - 原因は、底板を溝に入れたことで足元に80mm使ってしまう点。**溝の位置が実物では違う可能性が高いので、まず実測して割り付け直すこと。**
  - どうしても頭上を稼ぎたい場合は **底板を省いて道具箱を床置きにする**（オプション参照）。棚板1を160の溝に下げられ、シュレッダー段が702mmに戻る。
- **底板が床から80mm浮く**: 最下段の溝の位置による。床からの湿気やホコリが入りにくい利点がある一方、**扉で足元まで覆わないと隙間が正面から見える**。
- **扉 494×900×15、247mm幅の2枚接ぎ**: 外かぶせなので幅は外寸494mmと同じ。1820×250×15 を1枚買い、900×250を2枚取って **247mmにリップ**（3mmずつ落とす）。v1（238mm）よりリップ量が小さく、材料の使い方はさらに良い。
- **扉は床から900mmを覆う**: 底板が40mm浮いているので、扉を床まで下ろすと支柱の足元20mmと底板下の空間が隠れる。見た目が締まるうえ、床からのホコリの侵入も防げる。
- **扉の裏桟 380 × 40 × 18 を3本**: v1と同じ理屈で、**内寸440より短くしないと桟が支柱にぶつかって扉が浮く**。v2では棚板の余りから取る都合で380mmにしている（左右30mmずつの逃げ）。
  - 高さは **120〜160 / 450〜490 / 800〜840**。棚板（80〜98 / 240〜258 / 880〜898）と干渉しない位置に置く。

### 強度について

- **溝による支持は、ネジ止めより強い。** v1では棚板の小口に45mmネジを3本打って側板に留めていた。v2では棚板が溝に15mm差し込まれ、**60mm幅にわたって面で受ける**。さらに各溝には最初からビス穴が2つ空いているので、そこから32mm以上のビスで留めれば抜け止めになる。荷重の伝わり方として、こちらのほうが素直で強い。
- **棚板スパンは440mm** と短く、18mm集成材なら10kg程度でもたわみは目視できない。シュレッダー（4〜6kg）、道具箱（5kg前後）はいずれも余裕。
- **⚠ 横揺れ（ラッキング）対策は v1 より重要。** v1には400×900の側板が2枚あり、面材として横揺れをかなり抑えていた。**v2はそれがなくなるので、抵抗しているのは「支柱と棚板の溝接合」＋「背面の桟3本」だけ**になる。
  - **背面の桟3本は絶対に省略しないこと。** 後ろの支柱2本をつなぐ唯一の部材。
  - **各溝のビス穴（2つ）は全部使うこと。** 差し込んだだけでは接合が回転してしまう。ビスを入れて初めて剛性が出る。
  - **側面パネルを付ければさらに硬くなる**（下記オプション）。ぐらつきが気になったら後付けできる。
- **蝶番の取り付けが v1 より確実になった。** v1では18mm厚の側板の小口にネジを打つ必要があり、これが設計上の弱点だった。**v2では前面の支柱（27mm厚 × 60mm幅）に留められる**ので、ネジの効きが9mm分深くなり、面積にも余裕がある。扉の垂れ下がりに対する不安が実質的に消える。
- **扉の重量は約3.8kg**（板3.3kg + 裏桟0.5kg）。平蝶番3個。
- **転倒対策**: 幅494に対し高さ900（比率1:1.8）で倒れにくいが、扉を開けると重心が前に出る。壁側に寄せて設置する。背面の桟を通して壁の下地にL字金具で固定できる。

### 組み立て方法

1. **支柱の溝の位置を実測する。** 買ってきたら、まず床からの各溝の高さを測って書き出す。**この実測値で段の高さが決まる**ので、ここが設計の起点になる。目標は 底板80 / 棚板1 240 / 天板880（最下段の溝が80mm・ピッチ80mmの想定）だが、実物の溝に合わせて選び直す。**優先順位は「①道具箱が入る底段（≧120mm）→ ②シュレッダーの頭上をできるだけ広く」。**
2. **棚板の幅を確認する。** 470mmで切ってもらった板が、左右の溝にきちんと入るか1枚で試す。きつければ紙やすりで小口を落とす。**この確認を先にやること** — 3枚とも切ってから合わないと分かると手遅れになる。
3. **全部品の角と小口を紙やすりで整える**（#180 → #240）。玄関正面なので扉の角と天板の前縁は特に丸める。
4. **片側の支柱2本（前・後）に棚板3枚を差し込む。** 溝にボンドを少量塗り、棚板を差す。次にもう片側の支柱2本を差し込んで箱にする。**平らな床の上で作業する。**
5. **各溝のビス穴からビスを打つ。** 32mm以上の木ネジ。**溝1箇所につき2本、棚板3枚 × 4本の支柱 = 24本**。ここを省くと接合が回転して横揺れの原因になる。
6. **直角を確認する。** 対角線の長さを2方向測って同じになるよう調整する。
7. **背面の桟を3本取り付ける。** 後ろの支柱2本の間に、上（天板のすぐ下）・中（床から約500mm）・下（底板のすぐ上）へ角材を渡し、30mm木ネジで固定。**この工程で箱の剛性が決まる。**
8. **シュレッダーを実際に入れて確認する。** 扉を付ける前に、**実際の紙を持って上から投入する動作**を試す。合わなければ **棚板1を別の溝に移すだけで直せる** — これが v2 の最大の利点。v1では棚板を外して穴を開け直す必要があった。
9. **扉のパネルを作る。** 247×900の板2枚を平らな床に裏返しで並べ、木端にボンドを塗って突き合わせる。裏桟3本（380×40×18）を床から 60〜100 / 400〜440 / 800〜840 の高さに渡し、ボンド＋30mm木ネジで留める（1本の桟につき4本、継ぎ目をまたぐ配置）。**扉板は15mmなのでネジが表に突き抜けないよう下穴と長さを確認。** 重しを載せて1時間乾燥。
10. **扉を当ててみる。** 蝶番を付ける前に、閉じ位置に手で当てて**裏桟が棚の中にすっと収まるか**を確認する。
11. **扉に蝶番を付ける。** 平蝶番3個を、上端から100mm・中央・下端から100mmの位置で、**前面の支柱（27mm厚）**に留める。v1と違い小口ではなく支柱の面に打てるので、下穴を開ければ確実に効く。扉側は15mm厚なので**12mm程度の短いネジ**を使う。**吊り元は掃除機ステーションと反対の右側。**
12. **マグネットキャッチ2個と取っ手を付ける。**
13. **設置して動作確認。** シュレッダーが実際に使えるか、コードが背面から無理なく出ているかを確認する。**シュレッダーと道具箱は、裏桟の出っ張り（18mm）を避けて前面から18mm以上奥に置く。**

### オプション

- **側面パネル（推奨）**: 5.5mmシナ合板を支柱の外側にビス留めする。**ホコリの侵入を防ぎ、横揺れにも効き、見た目も締まる。** 左側は掃除機ステーション側から、右側は柱側から見えるかどうかで判断する。**構造材ではないので後付けできる** — まず組んでみて、ぐらつきや見た目が気になったら足せばよい。
- **底板を省いて道具箱を床置きにする（頭上を稼ぎたい場合）**: 底板を使わず、棚板1を **160の溝**に入れる。底段が床〜160mmになって道具箱（100mm）は余裕で入り、**シュレッダー段が622→702mm、頭上が292→372mm に戻る**。集成材も棚板2枚分で済む。引き換えに、道具箱が玄関の床に直接触れる。**頭上が足りないと分かったときの第一の対処法。**
- **可動棚の追加**: 支柱の溝は余っているので、**棚板を買い足せばいつでも段を増やせる。** 1820×400×18 の端材400×400から470幅は取れないので、増やすなら別途調達が要る点だけ注意。
- **転倒防止**: 背面上部の桟を通して壁の下地にL字金具で固定する。
- **底のフェルト/滑り止め**: 支柱の足元4箇所に貼る。**v2は接地面が4本の支柱だけ**なので、v1（側板2枚が線で接地）より床を傷つけやすい。**v1より重要度が高い。**
- **塗装**: 支柱はラジアータパイン無塗装、棚板はメルクシパインで**樹種が違うため色味が揃わない**。気になるならオイルステインで色を合わせる。扉は15mmと薄いので**片面だけ塗ると反る。必ず両面塗る。**
- **配線**: 電源タップはシュレッダー段の右奥。コードは背面の開口から出す。

- **パイン棚柱の900mm品の在庫と価格。** 本設計は1本1,100円（DCM同等品の実売）で見積もっている。ビバホーム品の価格は未確認。**4本必要なので、単価が500円違うと総額が2,000円動く。**
- **溝のピッチと最上・最下の溝の位置。** 約80mmピッチを前提にしているが未確認。段の高さはこれで決まる。
- **片溝か両溝か。** 本設計は片溝（溝が内側だけ）を前提にしている。両溝を買うと外側にも溝が出て見た目が変わる。
- **シュレッダーの奥行き。** 高さ33cmは確認済み。奥行きは未確認で、設計は400mm以内が前提。ダストボックスを手前に引き出すタイプなら、扉を開けた状態での引き出しスペースも要確認。
- **床から飾り棚下面までの1060mm。** 左右で床が傾いていることがある。

### どちらを選ぶか

**v2 を選ぶ理由になるのは「可動棚」と「組み立ての簡単さ」で、コストではありません。**

いま最大の未解決リスクは **シュレッダーの投入口の高さが実際に使えるか** です。v1 は棚板を固定するので、これを外すと棚板を外して開け直す作業になります。v2 なら棚板を別の溝に移すだけで済みます。**シュレッダーを実測せずに作り始められる**のは実際的な価値があります。

一方 **v1 の側板は、横揺れとホコリに対して確実な仕事をしています。** v2 でそれを取り戻すには側面パネルを足すことになり、そうすると価格差はなくなります。

- **確実さ・剛性・ホコリ対策を優先** → v1
- **調整のしやすさ・作りやすさを優先** → v2（側面パネルも付けるのが無難）
